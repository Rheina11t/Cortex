## Renewal Process
1. The ICO will send a reminder to the registered email address approximately 30 days before expiry.
2. Log in to the [ICO website](https://ico.org.uk/for-organisations/data-protection-fee/renew/) using the registration reference and the security code provided in the reminder.
3. Pay the annual fee (currently £40 for small organisations, or £35 if paid by Direct Debit).

## Updating Details
If the nature of data processing changes significantly (e.g., storing health data or criminal records), the registration should be updated on the ICO website.
